# README

## Installation Instructions


* Download:
```
git clone https://github.com/ericschoettle/Marios
cd Marios
```

* Install Gems
```
bundle install
```

* Start and configure Database
```
postgres
rails db:create db:migrate db:seed
```

* Start Web server  
```
rails s
```

* Navigate to 'localhost:3000' in your web browser, and enjoy!
